# Image Assets

Public image hosting for AI image generation and other projects.

## Usage

All images in this repository are publicly accessible at:

```
https://images.newbold.cloud/[folder]/[filename]
```

### Examples

- `https://images.newbold.cloud/references/my-image.png`
- `https://images.newbold.cloud/logos/patty-shack.png`
- `https://images.newbold.cloud/textures/wood-grain.jpg`

## Folder Structure

```
/public
  /references    - Reference images for AI generation
  /logos         - Brand logos and icons
  /textures      - Textures and patterns
  /photos        - General photos
  /assets        - Misc assets
```

## Supported Formats

- PNG (recommended for logos/graphics)
- JPG/JPEG (recommended for photos)
- WebP (modern format, smaller files)
- GIF (for animations)
- SVG (for vector graphics)

## AI Tool Compatibility

These URLs work with:
- Midjourney (paste URL in prompt)
- DALL-E / ChatGPT
- Stable Diffusion (img2img)
- Google Gemini
- Replicate
- Any tool that accepts image URLs
